Terminate EC2 instance
--
why to terminate
Terminating an Amazon EC2 instance permanently deletes the virtual machine, stopping all charges for compute capacity and releasing resources like Elastic IPs.

Implementation checklist:
--
1) To delete terminated EC2 Instances, locate the instance you want to delete in the EC2 Console under the Instances page.
2) Click on the instance and select the Terminate option.
3) Once you've selected “terminate,” you will receive a pop-up confirmation.
4) Click “Yes” to confirm.

Note:
It is primarily done to eliminate costs for unused resources, remove failed or unhealthy instances, or finalize automated scaling actions.
