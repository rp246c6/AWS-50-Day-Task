Create Volume Snapshot
--
Why to create Volumn Snapshot
Creating a volume snapshot involves generating a point-in-time, 
read-only copy of data from a storage volume to ensure data backup or state preservation.

Checklist for Implementation
1) Go to the EC2 Console 
2) select Volumes
3) Add tags and a description before finalization
4) Create Snapshot  > Actions > Create Snapshot.

Note:
A volume snapshot is a logical read-only point-in-time copy of a volume
