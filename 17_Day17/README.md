Create IAM Group
--
why to create IAM group:
an IAM Group is a collection of IAM users that allows 
you to manage permissions for multiple users at once, rather than individually.

Checklist to implement:
1) Sign in to the AWS Management Console and open the IAM console.
2) In the navigation pane, choose User groups and then select Create group.
3) User group name: Type a unique name for the group (e.g., Developers or Admins).
4) Choose Create group.
