Setting Up an EC2 Instance with an Elastic IP for Application Hosting
--
why to set up EC2 with Elastic IP:
Setting up an Amazon EC2 instance with an Elastic IP (EIP) ensures a 
static public IP address for your application, which is essential for consistent connectivity.

Checklist for implementation:
--
1) Launch an EC2 Instance.
2) Allocate an Elastic IP Address.
3) Associate the Elastic IP with Your Instance
