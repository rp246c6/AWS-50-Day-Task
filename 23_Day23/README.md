Data Migration Between S3 Buckets Using AWS CLI
--
why to migarate data between s3:
Migrating data between Amazon S3 buckets is done for business and technical reasons, 
including cost optimization, disaster recovery and compliance, and performance efficiency

Checklist for implementation:
1)  Create the New S3 Bucket.
2)  Sync All Data From the Source Bucket to Existing bucket.
3)  Verify the Data Migration.
