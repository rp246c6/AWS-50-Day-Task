Setting Up an EC2 Instance and CloudWatch Alarm
--
why to set up Ec2 Instance and cloudwatchAlarm.
--
Setting up an Amazon EC2 instance with CloudWatch Alarms is essential as it monitors key metrics like CPU 
or memory utilization, triggering automatic alerts or actions—such as restarting a failed instance or 
sending notifications—before issues cause downtime.

checklist for implementation:
1) Launch the EC2 Instance.
2) Create the CloudWatch Alarm.
3) Verify Configuration.
