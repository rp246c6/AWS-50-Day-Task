Create Key Pair
--

Why you need Key Pair in AWS:
--
You need to create a key pair in AWS primarily to securely authenticate and connect to your EC2 instances

Checklist for implementation
--
1) Open the EC2 console and choose Key Pairs under Network & Security.
2) Click Create key pair, enter a name, and select RSA (common) or ED25519.
3) Choose the format: .pem for OpenSSH/Linux/macOS or .ppk for PuTTY/Windows.
4) Click Create key pair to automatically download the private key file.
